1、常用的DOS命令
	1.1、（1）mkdir 创建新目录；
		（2）{
			cd 英文:change directory（改变目录）
			cd 加路径
			cd ..	返回上一路径
			cd \ 回到跟目录
			}
		（3）cls 清屏
		（4）dir 查看当前目录下有何东西
		（5）TAB键自动补全命令
	1.2、路径在windows系统中包括：相对路径和绝对路径。
		绝对路径：
			在Windows操作系统中凡是起点是盘符的都是绝对路径。
			eg：C:\Users\SOMEBODY> （绝对路径）
		相对路径：
			在路径形式上来看，相对路径绝对不是以盘符开始的。
			相对路径一定是相对于当前所在“位置”而言的。
		相对路径可以转绝对路径
